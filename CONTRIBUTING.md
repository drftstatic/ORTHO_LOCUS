# Contributing to ORTHO_LOCUS

## Project Context

ORTHO_LOCUS is a **private art installation** commissioned for [Rick Dailey](https://rickfdailey.com) and built by [Fladry Creative](https://fladrycreative.com).

While the codebase is open-source and available for educational and reference purposes, this is **not a public-facing product or service**. It exists as a commissioned artwork and technical exploration.

## Code Availability

The code is shared openly for:
- **Educational purposes** - Learn from the implementation
- **Reference** - See how location intelligence and AI can be combined
- **Inspiration** - Explore creative applications of mapping and generative AI

## Contributions

As this is a commissioned art project with specific artistic intent, we are **not accepting external contributions** at this time.

However, you are welcome to:
- Fork the repository for your own projects
- Use the code as a learning resource
- Adapt concepts for your own creative work
- Report critical bugs or security issues via email to [robb@fladrycreative.com](mailto:robb@fladrycreative.com)

## Questions or Inquiries

**About the artwork:** Contact Rick Dailey via [rickfdailey.com](https://rickfdailey.com)  
**Technical questions:** Contact Fladry Creative via [fladrycreative.com](https://fladrycreative.com)
